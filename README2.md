# Matrix Inverse Calculator GUI

## Description

This is a Tkinter-based graphical user interface (GUI) application for calculating matrix inverses. The application provides an intuitive interface where users can input square matrices of various sizes and compute their inverses with a single click.

## Features

- **Matrix Size Selector**: Choose matrix dimensions from 2×2 to 5×5 using a convenient spinbox control
- **Input Grid**: Interactive grid layout for entering matrix values
- **Compute Inverse Button**: Calculate the inverse of the input matrix
- **Clear Button**: Reset the interface and clear all inputs
- **Results Display**: View the computed inverse matrix in a dedicated results area
- **Resizable Window**: Flexible layout that adapts to different window sizes

## Requirements

- Python 3.x
- tkinter (included with standard Python installation)
- numpy

## How to Run

To launch the Matrix Inverse Calculator GUI, execute the following command:

```bash
python matrix_inverse_gui.py
```
